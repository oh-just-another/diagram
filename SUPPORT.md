# Support

Thanks for using the project! Here is where to get help.

## Documentation

Start with the docs and live examples at
**[ohjustanother.site](https://ohjustanother.site)**.

## Questions and help

- **Bugs and feature requests** — open a
  [GitHub issue](https://github.com/oh-just-another/diagram/issues) using the
  appropriate template. Please search existing issues first.
- **Questions, ideas, show-and-tell** — use
  [GitHub Discussions](https://github.com/oh-just-another/diagram/discussions).

## Security

For security vulnerabilities, **do not** open a public issue — follow the
[Security Policy](./SECURITY.md) instead.

## Note

This is a community, MIT-licensed project maintained on a best-effort basis —
there is no commercial support or SLA. Contributions and help answering others'
questions are always welcome.
